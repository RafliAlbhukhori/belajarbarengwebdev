Dasar-dasar HTML

HTML (Hypertext Markup Language) adalah dasar dari hampir semua halaman web yang Anda lihat. Ini adalah bahasa yang digunakan untuk membuat struktur dan konten dasar dari halaman web. Ketika Anda membuka browser web dan mengunjungi situs web apa pun, Anda sedang melihat HTML.

HTML adalah bahasa markup yang berarti Anda menggunakan tag untuk memformat dan mengorganisasi konten di halaman web. Beberapa tag dasar yang Anda gunakan dalam HTML termasuk `html`, `head`, `title`, `body`, `h1`, `p`, dan banyak lainnya. Dengan tag-tag ini, Anda dapat membuat tajuk, paragraf, daftar, tautan, gambar, dan banyak elemen lainnya.

Sebagai pemula dalam dunia HTML, Anda akan mempelajari dasar-dasar HTML terlebih dahulu. Anda akan belajar tentang tag-tag dasar, seperti `html`, `head`, dan `body`, yang membentuk kerangka dasar setiap halaman web. Anda juga akan memahami penggunaan tag judul `title` yang menunjukkan judul halaman web yang muncul di tab browser.

Selain itu, Anda akan mempelajari cara membuat tajuk dan paragraf menggunakan tag seperti `h1` dan `p`. Anda akan belajar cara menambahkan gambar dengan tag `img` dan membuat tautan dengan tag `a`. Anda juga akan mengeksplorasi cara membuat daftar dengan tag `ul`, `ol`, dan `li`.

Selama perjalanan belajar Anda, Anda akan menemukan berbagai atribut yang dapat Anda tambahkan ke tag HTML untuk mengubah tampilan dan perilaku elemen-elemen tersebut. Anda akan memahami cara mengatur warna latar belakang, ukuran teks, dan berbagai atribut lainnya untuk menyesuaikan halaman web Anda.

Dengan pemahaman dasar-dasar HTML, Anda akan memiliki landasan kuat untuk membangun dan merancang halaman web sederhana. Ini adalah langkah pertama yang penting dalam perjalanan Anda menuju menjadi seorang pengembang web yang kompeten. Terus berlatih, eksperimen dengan kode, dan jangan ragu untuk menggali lebih dalam ke dalam HTML serta teknologi web lainnya. Selamat belajar!

Cara Mengatur Font Di HTML

Mengatur font dalam HTML adalah salah satu aspek penting dalam desain dan tampilan sebuah halaman web. Anda dapat menggunakan tag `font` untuk mengatur berbagai aspek font, seperti ukuran, jenis huruf, dan warna. Dengan pemahaman tentang cara mengatur font, Anda dapat membuat konten web Anda lebih menarik dan mudah dibaca.

Tag `font` dapat digunakan dengan beberapa atribut yang memungkinkan Anda untuk mengendalikan tampilan teks. Salah satu atribut utama adalah `size`, yang digunakan untuk mengatur ukuran font. Misalnya, `font size="4"` akan membuat teks menjadi ukuran font 4. Anda dapat mengganti nilai atribut `size` dengan angka lain sesuai dengan preferensi Anda.

Selain ukuran font, Anda juga dapat mengubah jenis huruf menggunakan atribut `face`. Misalnya, `font face="Arial"` akan mengubah jenis huruf teks menjadi Arial. Anda bisa menyesuaikan jenis huruf sesuai dengan keinginan Anda.

Selanjutnya, Anda dapat menggunakan atribut `color` untuk mengatur warna font. Misalnya, `font color="red"` akan mengubah warna teks menjadi merah. Anda dapat mengganti nilai atribut `color` dengan kode warna yang diinginkan, seperti "blue", "green", atau kode warna heksadesimal.

tag `font` dan atribut-atributnya, Anda memiliki fleksibilitas dalam mengatur tampilan teks di halaman web Anda. Namun, seiring berjalannya waktu, pendekatan ini telah menjadi usang, dan lebih disarankan untuk menggunakan CSS (Cascading Style Sheets) untuk mengatur tampilan font dan desain halaman web secara keseluruhan. CSS memberikan kontrol yang lebih kuat dan semakin umum digunakan dalam pengembangan web modern.

Cara Menambahkan Gambar

Menambahkan gambar ke halaman web adalah cara yang efektif untuk memperkaya konten dan membuat halaman web lebih menarik. Anda dapat menggunakan tag `img` dalam HTML untuk menampilkan gambar.

Untuk menambahkan gambar, cukup gunakan tag `img` dan atribut `src` untuk menentukan sumber gambarnya. Atribut `src` harus berisi alamat file gambar yang ingin Anda tampilkan. Misalnya, `img src="gambar1.jpg"` akan menampilkan gambar dengan nama "contoh.jpg" yang terletak di direktori yang sama dengan halaman HTML Anda.

Selain atribut `src`, Anda dapat menggunakan atribut `alt` untuk memberikan teks alternatif untuk gambar. Ini adalah teks yang akan ditampilkan jika gambar gagal dimuat atau jika pengguna menggunakan perangkat pembaca layar. Teks alternatif ini juga bermanfaat untuk SEO.

Anda juga dapat mengontrol tinggi dan lebar gambar menggunakan atribut `width` dan `height`. Misalnya, `img src="gambar2.jpg" width="200" height="150"` akan menampilkan gambar dengan lebar 200 piksel dan tinggi 150 piksel. Ini memungkinkan Anda mengatur ukuran gambar sesuai dengan preferensi Anda.

Gambar dapat digunakan untuk memperkuat pesan atau mengilustrasikan konsep, sehingga membuat pengalaman pengguna lebih menarik.
